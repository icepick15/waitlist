# Waitlist — Service Provider Management Dashboard

A React-based dashboard for managing service providers, built with Create React App and deployed on Vercel.

## Features

- Service provider listing with sorting and pagination
- Global search and column filtering
- Edit service provider details via modal
- Toast notifications for user feedback
- Responsive design (mobile & desktop) with Tailwind CSS

## Tech Stack

- React 19
- Tailwind CSS
- TanStack Table (react-table v8)
- Lucide React (icons)
- date-fns

## Getting Started

### Install dependencies

```bash
npm install
```

### Run in development

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser. The page reloads on changes.

### Build for production

```bash
npm run build
```

Outputs to the `build/` folder, minified and optimised.

### Run tests

```bash
npm test
```

## Deployment

This project is deployed via [Vercel](https://vercel.com). Push to the `main` branch to trigger a new deployment.
